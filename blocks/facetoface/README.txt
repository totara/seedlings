Description
------------

This block complements the Face-to-face module by providing site-wide listings
of all sessions and user bookings.


Requirements
-------------

* Moodle 2.2
* Face-to-face module


Installation
-------------

1- Unpack the module into your moodle install in order to create a
   blocks/facetoface directory.

2- Visit the /admin/index.php page to trigger the database installation.
